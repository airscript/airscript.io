# Airscript Frontend

# Developers

## Requirements

- Ruby 
- RubyGems
- Bundler
- Git
- Middleman
- Foreman

### Setup 

Run `foreman start -p 4567` to boot up the server on whichever port you like.

### Deploy

This will likely be deployed on a GitHub page. When it gets to that point, deploys will be as simple as `bundle exec rake deploy`
